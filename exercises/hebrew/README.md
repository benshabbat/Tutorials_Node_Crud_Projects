## איך לתרגל?

1. **קרא את התרגיל** - הבן מה מבוקש
2. **נסה לבד** - תכתוב את הקוד לפני שאתה מסתכל על הפתרון
3. **הרץ ובדוק** - וודא שזה עובד
4. **השווה** - בדוק את הפתרון אחרי שסיימת
5. **שפר** - נסה להוסיף תכונות נוספות

## טיפים להצלחה

✅ **התחל קטן** - תרגיל אחד בכל פעם

✅ **קרא שגיאות** - הן אומרות לך מה לא בסדר!  
✅ **התנסה** - נסה ערכים שונים
