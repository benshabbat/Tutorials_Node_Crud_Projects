# 🚀 Projects - פרויקטים

## Available Projects

### [modular-crud/](../modular-crud/)
**CRUD מודולרי מובנה**

פרויקט CRUD מאורגן עם מבנה נכון:
- **Controllers** - לוגיקה של endpoints
- **Routes** - הגדרת נתיבים
- **Models** - מבנה הנתונים
- **Middleware** - טיפול בשגיאות
- **Error Handling** - טיפול מקצועי בשגיאות

**מומלץ למתחילים שמבינים את היסודות!**

---

### [modular-with-services/](../modular-with-services/)
**CRUD עם Services Layer**

פרויקט מתקדם עם הפרדה מלאה:
- **Controllers** - טיפול בבקשות/תגובות
- **Services** - לוגיקה עסקית
- **Models** - מבנה נתונים
- **Routes** - ניתוב
- **Middleware** - שכבות ביניים

**מומלץ למי שמבין מבנה מודולרי!**

## איך להתחיל עם פרויקט?

```bash
# בחר פרויקט
cd modular-crud

# התקן תלויות
npm install

# הרץ את הפרויקט
npm start

# או במצב פיתוח (עם auto-reload)
npm run dev
```

## מה ללמוד מכל פרויקט?

### Modular CRUD
✅ איך לארגן קוד  
✅ הפרדה בין שכבות  
✅ Controllers, Routes, Models

### Modular with Services
✅ Service Layer Pattern  
✅ הפרדה מלאה של אחריות

✅ ארכיטקטורה ניתנת להרחבה  
✅ קוד נקי וניתן לבדיקה
